

 O que é Frontend?

Frontend é a parte visual de um sistema ou site com a qual o usuário interage diretamente. 
Tudo o que aparece na tela, como botões, menus, formulários, imagens e textos, faz parte do frontend.

 O que é Cliente x Servidor?

O modelo Cliente-Servidor é uma forma de comunicação entre computadores:

- Cliente: é quem faz a solicitação. Normalmente é o navegador (Chrome, Firefox, Edge etc.) usado pelo usuário.
- Servidor: é o computador ou sistema que recebe a solicitação, processa as informações e envia uma resposta ao cliente.

Exemplo: quando você acessa um site, seu navegador (cliente) solicita a página ao servidor, que envia os arquivos necessários para exibição.

 Qual é o papel de HTML, CSS e JavaScript?

HTML (HyperText Markup Language)
Responsável pela estrutura e conteúdo da página.
Exemplo: títulos, parágrafos, imagens, links e formulários.

 CSS (Cascading Style Sheets)
Responsável pela aparência visual da página.
Exemplo: cores, fontes, espaçamentos, tamanhos e layout.

JavaScript
Responsável pela interatividade e comportamento da página.
Exemplo: validação de formulários, animações, menus dinâmicos e atualizações de conteúdo sem recarregar a página.

 O que é W3C?

W3C (World Wide Web Consortium) é uma organização internacional que desenvolve e mantém os padrões da Web.
